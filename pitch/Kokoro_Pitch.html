LGPD
Lei Geral de Proteção de Dados
Brasil — Lei 13.709/2018, Art. 5°, II
Dados de saúde são dados sensíveis pela LGPD e exigem tratamento especial, com risco de discriminação se mal gerenciados.
O Kokoro segue
Criptografia de ponta a ponta · Consentimento explícito · Dados mínimos · DPO designado
Obrigatoriedades LGPD
Consentimento explícito: autorização clara do titular para uso dos dados sensíveis.
Finalidade específica: dados usados apenas para objetivos claramente definidos.
Minimização de dados: coletar apenas o estritamente necessário.
Segurança da informação: controles técnicos para evitar acessos não autorizados.
Accountability: comprovar que a organização cumpre a LGPD em seus processos.
